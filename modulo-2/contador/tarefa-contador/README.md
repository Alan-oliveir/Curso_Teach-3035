## Tarefa: Contador com React + TypeScript

Este projeto foi desenvolvido como parte das tarefas do módulo 2 do curso.   
O objetivo principal foi criar uma aplicação simples que gerencia o estado de um contador de cliques.  

### Sobre a Tarefa
A tarefa consistia em:
1. Criar uma tela com um botão.
2. Exibir a quantidade de vezes que o usuário clicou no botão.

## Tecnologias e Ferramentas
* **React** (Biblioteca principal)
* **TypeScript** (Tipagem estática para maior segurança)
* **Vite** (Ferramenta de build e servidor de desenvolvimento)

> **Nota:** Para o desenvolvimento desta tarefa, utilizei como base o código inicial gerado pelo comando `npm create vite@latest`. Fiz modificações no componente principal para adaptar o contador aos requisitos da tarefa e personalizei os logos e textos conforme necessário.
